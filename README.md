# Game (Boy) of Life

Source code of [Game (Boy) of Life](https://studioguma.itch.io/game-boy-of-life), my Game Boy port of John Conway’s Game of Life.

All code and assets by me. Made with [GBDK 2020](https://github.com/gbdk-2020/gbdk-2020), [GBTD](https://github.com/gbdk-2020/GBTD_GBMB), and [hUGETracker](https://github.com/SuperDisk/hUGETracker).

Building requires GBDK 2020, as well as Git or GNU Make on Windows. Make sure to edit the makefile based on your OS and the path you have GBDK in. To build, open a command line and type `make`, and type `make clean` to clean. The compiled ROM can be played on an emulator or flashcart.

ROM size: 32 KB
